# 🥣 Fuchka Frenzy 🌶️

**Fuchka Frenzy** is a fast-paced, addictive Android arcade game where you step into the shoes of a street food maestro! Serve the perfect plate of Fuchka (Pani Puri) to your hungry customers by matching their specific ingredient requests before the timer runs out.

---



## ✨ Features

- **Dynamic Ordering System:** Every customer wants a unique combination of 3 to 5 ingredients.
- **Time-Trial Challenge:** Start with 30 seconds and fight to keep the clock ticking.
- **Combo Bonuses:** Earn **+10 points** and **+2 seconds** for every correct plate served.
- **High Stakes:** A single mistake costs you **-3 seconds**!
- **Modern UI:** Built entirely with **Jetpack Compose** for a smooth, reactive user experience.
- **Vibrant Visuals:** Uses colorful emojis and intuitive design to represent traditional ingredients.

---

## 🛠️ Tech Stack

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Architecture:** MVVM (Model-View-ViewModel)
- **State Management:** Compose State & ViewModel
- **Concurrency:** Kotlin Coroutines for the game timer logic.
- **Build System:** Gradle (Kotlin DSL)

## 🚀 Installation & Setup

To get a local copy up and running, follow these simple steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/bigiq/Fuchka-Frenzy.git
   ```
2. **Open in Android Studio:**
   - Launch Android Studio (Ladybug or newer recommended).
   - Select **File > Open** and navigate to the cloned directory.
3. **Sync Project:**
   - Wait for the Gradle sync to complete.
4. **Run the App:**
   - Connect an Android device or start an emulator.
   - Click the **Run** button (green play icon) in the toolbar.

---

## 🎮 How to Play

1. **Observe the Order:** Look at the top of the screen to see what the customer wants (e.g., 🥔 + 🌶️ + 🧅).
2. **Prepare the Tray:** Tap the ingredient buttons at the bottom to add them to your tray.
3. **Automatic Check:** Once your tray matches the order length, the game will automatically check your work.
4. **Be Fast:** Keep serving correctly to gain bonus time and achieve the highest score possible!
